# Git CheatSheet

## About:
This is a collection of tips and tricks on using Git and its' related tools.


Git
----
Checking the Stat of Branches and Remote

* ```git branch -vv```
* ```git remote show origin```

* ```git pull --all```
* Delete a remote branch: ```git push origin :branch_to_remove```
